Issue Deep-Dive

The issues I encountered were with the cursor image, flex box content, and event listener. For the cursor, the image would not load properly. First, I wanted to know if the image was the problem or if the line of code had an issue. To do so, I set the fallback cursor keyword to "help". So when the cursor changed to the "help", I knew that it was because the specified image failed to load. Then, I made sure to properly declare the path to the image (cursors/cursor-name.png). After further reseach, I realized the issue was with the size of the image. It is recommended to use icons that are at most 32x32 px. Once I changed the size, the image was able to load properly.
