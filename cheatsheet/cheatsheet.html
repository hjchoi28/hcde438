Cheatsheet

HTML

  • Encoded Characters
    • a tab is <& ensp;> without the space
    • a line break is <br>
    • "<" is <& lt;> without the space
    • "<" is <& ensp;> without the space
    • ">" is <& gt;> without the space
  • DOCTYPE!

    • Always declare "<!DOCTYPE html>" at the top of a html file.
    It is standard practice and it removed my issue with collapsing top margin for the header.

Javascript

  • addEventListener

    • Format: element.addEventListener("action", function)

    Make sure the function name does not have a parenthesis! This will cause the function to be called regardless of whether the specified action occured.

  • Document Object

    • You can specify the element you want to change, get the value of, listen to, etc... by calling on its id. Ex. document.getElementId("id-name")

  • Functions

    • Format: function fName() {}

  • Variables
    • let varname = value that can be altered
    • const varname = value that stays constant
